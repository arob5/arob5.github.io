Regularized Least Squares with Singular Prior

Solving the regularized least squares optimization problem when the prior covariance matrix is not positive definite.

Published

03 December 2024

Setup

Recall the L2L^2L2 regularized least squares problem (i.e., ridge regression),

\begin{align} u_{\star} &:= \text{argmin}_{u \in \mathbb{R}^d} J(u) \tag{1} \newline J(u) &:= \frac{1}{2} \lVert y - Gu\rVert^2_{\Sigma} + \frac{1}{2}\lVert u-m \rVert^2_{C} \end{align}

where y∈Rny \in \R^ny∈Rn and m∈Rdm \in \R^dm∈Rd are fixed vectors, G∈Rn×dG \in \R^{n \times d}G∈Rn×d is the linear forward model, and Σ∈Rn×n\Sigma \in \R^{n \times n}Σ∈Rn×n and C∈Rd×dC \in \R^{d \times d}C∈Rd×d are positive definite matrices. In (1) we have used the following notation for inner products and norms weighted by a positive definite matrix CCC: \begin{align} \langle v, v^\prime \rangle_C &:= \langle C^{-1}v, v^\prime\rangle = v^\top C^{-1}v^\prime \newline \lVert v \rVert_C^2 &:= \langle v, v\rangle_C. \end{align}

A solution to (1) can be viewed as a maximum a posteriori (MAP) estimate under the Bayesian model \begin{align} y|u &\sim \mathcal{N}(Gu, \Sigma) \tag{3} \newline u &\sim \mathcal{N}(m, C). \tag{4} \end{align}

I discuss this problem in depth in my post, on the linear Gaussian model, where I show that the solution can be written as the following equivalent expressions, \begin{align} u_{\star} &= \left(G^\top \Sigma^{-1}G + C^{-1}\right)^{-1}\left(G^\top \Sigma^{-1}y + C^{-1}m \right) \tag{5} \newline u_{\star} &= m + CG^\top \left(GCG^\top + \Sigma \right)^{-1}(y-Gm). \tag{6} \end{align}

In this post we will consider a generalization of problem (1), where CCC is only required to be positive semidefinite (not strictly positive definite). In particular, this means that the inverse C−1C^{-1}C−1 may not exist. It is interesting to note that, although (5) depends on C−1C^{-1}C−1, the inverse does not appear anywhere in the equivalent expression (6). This leads to the natural question of whether (6) still provides a valid solution to the optimization problem even when CCC is singular. The first step in investigating this question will be providing a suitable generalization of the objective function in (1), since the expression ∥u−m∥C2=∥C−1/2(u−m)∥2\lVert u - m\rVert^2_{C} = \lVert C^{-1/2}(u - m)\rVert^2∥u−m∥C2​=∥C−1/2(u−m)∥2 is not well-defined when CCC is singular.

Generalizing the Optimization Problem

Our goal is now to extend (1) to the case where CCC need not be strictly positive definite. We start by generalizing (1) through the definition of a constrained optimization problem on an extended parameter space. We then provide two approaches to formulate this problem in an unconstrained fashion, each of which leads to equivalent closed-from solutions. We will find it convenient to define notation for the “centered” quantities \begin{align} &u^\prime := u - m, &&y^\prime := y - Gm \tag{7} \end{align} so that y−Gu=(y−Gm)−G(u−m)=y′−Gu′.(8) y - Gu = (y - Gm) - G(u-m) = y^\prime - Gu^\prime. \tag{8} y−Gu=(y−Gm)−G(u−m)=y′−Gu′.(8)

Constrained Formulation

We start by considering the constrained formulation. This section follows the approach in section 8.1.2 of (Sanz-Alonso et al., 2023). To start, note that when CCC is positive definite, we have ∥u−m∥C2=∥u′∥C2=⟨C−1u′,u′⟩=⟨b,u′⟩,(9) \lVert u - m\rVert^2_C = \lVert u^\prime \rVert^2_C = \langle C^{-1}u^\prime, u^\prime\rangle = \langle b, u^\prime \rangle, \tag{9} ∥u−m∥C2​=∥u′∥C2​=⟨C−1u′,u′⟩=⟨b,u′⟩,(9) where b∈Rdb \in \mathbb{R}^db∈Rd solves Cb=u′.(10) Cb = u^\prime. \tag{10} Cb=u′.(10) When CCC is invertible, the unique bbb solving (10) is simply found by multiplying both sides by C−1C^{-1}C−1. When CCC is not invertible, then there may be zero or infinitely many such solutions. As long as there is at least one solution to (10) we can give meaning to expression (9) by picking a particular bbb that solves (10). Now that we have introduced a new variable bbb, we might consider generalizing (1) by jointly optimizing over (u,b)(u,b)(u,b) subject to the linear constraint Cb=u′Cb = u^\primeCb=u′.

Constrained Joint Optimization.
\begin{align} (u_{\star}, b_{\star}) &\in \text{argmin}_{(u,b) \in \mathcal{S}} J(u,b) \tag{11} \newline J(u,b) &:= \frac{1}{2} \lVert y - Gu\rVert^2_{\Sigma} + \frac{1}{2}\langle b, u-m \rangle \newline \mathcal{S} &:= \left\{(u,b) : Cb = u-m \right\}. \end{align}

Note that if CCC is positive definite, then (11) reduces to (1). If the problem is solved for (u⋆,b⋆)(u_{\star}, b_{\star})(u⋆​,b⋆​), then the desired solution can be obtained by extracting u⋆u_{\star}u⋆​ and discarding the nuisance parameter b⋆b_{\star}b⋆​.

Lagrange Multipliers

Observe that (11) can be viewed as optimizing J(u,b)J(u,b)J(u,b) subject to the constraint g(u,b)=0g(u,b) = 0g(u,b)=0, where g(u,b)=Cb−(u−m)g(u,b) = Cb - (u-m)g(u,b)=Cb−(u−m). This is a typical setup for Lagrange multipliers. We therefore introduce the Lagrange multiplier λ∈Rd\lambda \in \mathbb{R}^dλ∈Rd, which allows us to cast the constrained optimization over (u,b)(u,b)(u,b) as an unconstrained optimization over (u,b,λ)(u,b,\lambda)(u,b,λ).

Lagrange Multiplier Formulation.
\begin{align} (u_{\star}, b_{\star}, \lambda_{\star}) &\in \text{argmin}_{u,b,\lambda} J(u,b,\lambda) \tag{12} \newline J(u,b,\lambda) &:= \frac{1}{2} \lVert y - Gu\rVert^2_{\Sigma} + \frac{1}{2}\langle b, u-m \rangle + \langle \lambda, Cb - u + m \rangle. \end{align}

We have succeeding in converting the problem to one that can be solved by analytical means; namely, by solving the system of equations ∇uJ=∇bJ=∇λJ=0.(13) \nabla_{u} J = \nabla_{b} J = \nabla_{\lambda} J = 0. \tag{13} ∇u​J=∇b​J=∇λ​J=0.(13)

A solution (u⋆,b⋆,λ⋆)(u_{\star}, b_{\star}, \lambda_{\star})(u⋆​,b⋆​,λ⋆​) of (12), projected onto the uuu-component, is given by u⋆=m+Cb⋆,(14) u_{\star} = m + Cb_{\star}, \tag{14} u⋆​=m+Cb⋆​,(14) where b⋆=[(G⊤Σ−1G)C+I]−1G⊤Σ−1(y−Gm).(15) b_{\star} = \left[(G^\top \Sigma^{-1}G)C + I \right]^{-1}G^\top \Sigma^{-1}(y-Gm). \tag{15} b⋆​=[(G⊤Σ−1G)C+I]−1G⊤Σ−1(y−Gm).(15) It also holds that C[(G⊤Σ−1G)C+I]−1G⊤Σ−1=CG⊤(GCG⊤+Σ)−1,(16) C\left[(G^\top \Sigma^{-1}G)C + I \right]^{-1}G^\top \Sigma^{-1} = CG^\top \left(GCG^\top + \Sigma \right)^{-1}, \tag{16} C[(G⊤Σ−1G)C+I]−1G⊤Σ−1=CG⊤(GCG⊤+Σ)−1,(16) which implies that (14) agrees with expression (6).

Basis Approach

We now consider an alternative approach that avoids the joint optimization over (u,b)(u,b)(u,b). Similar exposition can be found in section 8.3 of (Evensen et al., 2022). Looking back to (9), instead of choosing to optimize over all bbb satisfying (10), we will instead simply choose a particular bbb satisfying this constraint. This only makes sense if the particular choice of bbb does not matter; i.e., if the value of ⟨b,u′⟩\langle b, u^\prime \rangle⟨b,u′⟩ is the same for any bbb solving (10). The below propostion shows that this is indeed the case.

Proposition.
Let u∈Rdu \in \mathbb{R}^du∈Rd be a vector such that there is at least one solution b∈Rdb \in \mathbb{R}^db∈Rd to Cb=u′Cb = u^\primeCb=u′. Then ⟨b,u′⟩\langle b, u^\prime \rangle⟨b,u′⟩ and J(u,b)J(u,b)J(u,b) are constant for any choice of bbb solving this linear system.

Proof. Let b,b′∈Rdb, b^\prime \in \R^db,b′∈Rd satisfy Cb=Cb′=u′Cb = Cb^\prime = u^\primeCb=Cb′=u′. It thus follows that ⟨b,u′⟩−⟨b′,u′⟩=⟨b−b′,u′⟩=⟨b−b′,Cb⟩=⟨C(b−b′),b⟩=⟨0,b⟩=0, \langle b, u^\prime\rangle - \langle b^\prime, u^\prime\rangle = \langle b - b^\prime, u^\prime\rangle = \langle b - b^\prime, Cb\rangle = \langle C(b - b^\prime), b\rangle = \langle 0, b\rangle = 0, ⟨b,u′⟩−⟨b′,u′⟩=⟨b−b′,u′⟩=⟨b−b′,Cb⟩=⟨C(b−b′),b⟩=⟨0,b⟩=0, where we have used the linearity of the inner product and the fact that CCC is symmetric. Since the inner product term in (11) is the only portion with dependence on bbb, it follows that J(v,b)=J(v,b′)J(v,b) = J(v,b^\prime)J(v,b)=J(v,b′). ■\qquad \blacksquare■

Thus, for each uuu that yields a consistent system Cb=u′Cb = u^\primeCb=u′, we can simply pick any solution bbb to insert into ⟨b,u′⟩\langle b, u^\prime\rangle⟨b,u′⟩. The objective will be well-defined since the above result verifies that the specific choice of bbb is inconsequential. A natural choice is to select the bbb of minimal norm; that is, b†:=argminCb=u′∥b∥.(17) b^{\dagger} := \text{argmin}_{Cb=u^\prime} \lVert b \rVert. \tag{17} b†:=argminCb=u′​∥b∥.(17) This unique minimal norm solution is guaranteed to exist and is conveniently given by the Moore-Penrose pseudoinverse b†=C†u′.(18) b^{\dagger} = C^{\dagger}u^\prime. \tag{18} b†=C†u′.(18) Note that when CCC is positive definite, C†=C−1C^{\dagger} = C^{-1}C†=C−1. We can now eliminate the requirement to optimize over bbb by considering the following optimization problem.

Pseudoinverse Formulation.
\begin{align} u_{\star} &\in \text{argmin}_{u \in \mathcal{S}} J(u) \tag{19} \newline J(u) &:= \frac{1}{2} \lVert y - Gu\rVert^2_{\Sigma} + \frac{1}{2}\langle C^{\dagger}(u-m), u-m \rangle \newline \mathcal{S} &:= \left\{u \in \mathbb{R}^d: u-m \in \mathcal{R}(C) \right\} \end{align}

Note that we are now required to reintroduce a constraint set S\mathcal{S}S. This is due to the fact that the pseudoinverse produces a solution bbb to Cb=u′Cb = u^\primeCb=u′ when a solution exists, but we must still explicitly restrict the search space to uuu that admit a consistent system Cb=u′Cb = u^\primeCb=u′; i.e., u′∈R(C)u^\prime \in \mathcal{R}(C)u′∈R(C). The previous approach dealt with this issue by introducing a Lagrange multiplier λ\lambdaλ. We now show that (18) can be written as an unconstrained problem without extending the parameter space. This relies on the following important fact.

Subspace Property.
Let A∈Rd×rA \in \mathbb{R}^{d \times r}A∈Rd×r be a matrix of rank r≤dr \leq dr≤d satisfying C=AA⊤C = AA^\topC=AA⊤. Then a solution to the optimization problem (19) is constrained to the rrr-dimensional affine subspace m+R(C)=m+R(A).(20) m + \mathcal{R}(C) = m + \mathcal{R}(A). \tag{20} m+R(C)=m+R(A).(20)

This is simply a rewriting of the requirement u′=u−m∈R(C)u^\prime = u-m \in \mathcal{R}(C)u′=u−m∈R(C). The solution is constrained by the rank of the subspace R(C)\mathcal{R}(C)R(C). If r=dr = dr=d (i.e., CCC is full-rank) then we recover the typical unconstrained least squares problem. In general, r<dr < dr<d so the columns of CCC are linearly dependent. We therefore introduce the matrix AAA whose columns a1,…,ara_1, \dots, a_ra1​,…,ar​ form a basis for the range of CCC; i.e., the columns of CCC and AAA have the same span.

To implicitly encode the constraint u′∈R(C)u^\prime \in \mathcal{R}(C)u′∈R(C), we can look for solutions in m+R(A)m + \mathcal{R}(A)m+R(A). Any vector in this space can be written as u=m+∑j=1rwjaj=m+Aw,(21) u = m + \sum_{j=1}^{r} w_j a_j = m + Aw, \tag{21} u=m+j=1∑r​wj​aj​=m+Aw,(21) for some weight vector w:=(w1,…,wr)⊤∈Rrw := (w_1, \dots, w_r)^\top \in \mathbb{R}^rw:=(w1​,…,wr​)⊤∈Rr. We can simplify this even further using the following properties of the pseudoinverse:

  1. By definition, the pseudoinverse satisfies A=A(A†A)A = A(A^{\dagger}A)A=A(A†A).

  3. The following identity holds: A†=A⊤(AA⊤)†A^{\dagger} = A^\top (AA^\top)^{\dagger}A†=A⊤(AA⊤)†.

We can thus simplify the second term in (22) as \begin{align} \langle (AA^\top)^{\dagger}Aw, Aw\rangle = \langle A^\top(AA^\top)^{\dagger}Aw, w\rangle = \langle (A^{\dagger}A)w, w\rangle &= \langle (A^{\dagger}A)^2w, w\rangle \newline &= \langle (A^{\dagger}A)w, (A^{\dagger}A)w\rangle \newline &= \lVert (A^{\dagger}A)w \rVert^2, \tag{23} \end{align} where the second equality uses the third pseudinverse property above. Unconstrained Pseudoinverse Formulation.
The optimization problem in (19) can equivalently be formulated as the unconstrained problem \begin{align} w_{\star} &\in \text{argmin}_{w \in \mathbb{R}^r} J(w) \tag{27} \newline J(w) &:= \frac{1}{2} \lVert y^\prime - GAw\rVert_{\Sigma}^2 + \frac{1}{2}\lVert w \rVert^2 \end{align}

One might initially take issue with the claim that (27) is actually unconstrained given that it relies on the re-parameterization (24), which constrains the weights to lie in the range of A†AA^{\dagger}AA†A. Proof. Observe that (27) is of the form (1) with u:=wu := wu:=w, y′:=yy^\prime := yy′:=y, G:=GAG := GAG:=GA, and C:=IC := IC:=I. Concluding Notes

We have successfully shown that the optimum (6) continues to hold for the generalization (11) of the least squares formulation, even when CCC is singular. We provided two different derivations of the solution, resulting in the expressions (14) and (29), both of which agree with u⋆u_{\star}u⋆​ as given in (6). We conclude with a few brief follow-up comments.

Uniqueness

Our derivations show that the optimum u⋆u_{\star}u⋆​ is unique, even when CCC is singular. Note that this does not imply that a solution (u⋆,b⋆)(u_{\star}, b_{\star})(u⋆​,b⋆​) to (11) is unique, since many choices of bbb may lead to the same value of the objective. Indeed, u⋆=m+Cb⋆u_{\star} = m + Cb_{\star}u⋆​=m+Cb⋆​ but it may be that u⋆=m+Cb′u_{\star} = m + Cb^\primeu⋆​=m+Cb′ for some other b′∈Sb^\prime \in \mathcal{S}b′∈S. n particular, if u⋆u_{\star}u⋆​ is optimal, then (u⋆,b′)(u_{\star}, b^\prime)(u⋆​,b′) minimizes J(u,b)J(u,b)J(u,b) for any b′b^\primeb′ satisfying Cb′=u⋆−mCb^\prime = u_{\star} - mCb′=u⋆​−m. This follows immediately from the result that the particular bbb solving this linear system does not change the value of the objective. Given that the particular choice of bbb is inconsequential, then a particular rule for choosing bbb will not affect the optimal u⋆u_{\star}u⋆​. The specific approach of choosing the minimal norm bbb led to the least squares problem (27), which we know has a unique solution. Therefore, if (11) has a solution, then it is unique. We summarize this below.

Uniqueness.
If the solution set S\mathcal{S}S is non-empty, then there is a unique optimal value u⋆u_{\star}u⋆​ solving (11).

Representations of the solution

Continuing the above discussion, it should also be emphasized that, while the solution is unique, there may be many different ways to represent it. To see this, consider that (14) gives the solution in the form u⋆=m+Cb⋆u_{\star} = m + Cb_{\star}u⋆​=m+Cb⋆​, for a weight vector b⋆b_{\star}b⋆​. But in general CCC has linearly dependent columns, and thus there may be multiple sets of weights that give rise to the same vector. In (29) the solution is instead represented in the form u⋆=m+Aw⋆u_{\star} = m + Aw_{\star}u⋆​=m+Aw⋆​. If we assume that the columns of AAA are linearly independent, thus providing a basis for R(C)\mathcal{R}(C)R(C), then this provides the unique representation of the solution with respect to the particular basis a1,…,ara_1, \dots, a_ra1​,…,ar​. Note that the two representations are connected by u⋆=m+Cb⋆=m+AA⊤b⋆=m+A(A⊤b⋆).(29) u_{\star} = m + Cb_{\star} = m + AA^\top b_{\star} = m + A(A^\top b_{\star}) \tag{29}. u⋆​=m+Cb⋆​=m+AA⊤b⋆​=m+A(A⊤b⋆​).(29) If the columns of AAA are independent, then (29) implies w⋆=A⊤b⋆w_{\star} = A^\top b_{\star}w⋆​=A⊤b⋆​.

Applications

Why should we even care about the case when CCC is singular? The main application that motivated this post comes from the field of data assimilation. In this context, the parameter and data dimensions ddd and nnn can be massive, potentially precluding storing the matrices Σ\SigmaΣ or CCC. A common solution to this problem is to replace the true CCC with a low-rank approximation of the form C=1r−1∑j=1r(u(j)−m)(u(j)−m)⊤,(30) C = \frac{1}{r-1} \sum_{j=1}^{r} (u^{(j)} - m)(u^{(j)} - m)^\top, \tag{30} C=r−11​j=1∑r​(u(j)−m)(u(j)−m)⊤,(30) where r<dr < dr<d. This Monte Carlo approximation results in a sample covariance matrix CCC that has rank at most rrr. Here, {u(j)}\{u^{(j)}\}{u(j)} is a set of rrr samples used to compute the sample covariance. If we define AAA to be the d×rd \times rd×r matrix with jthj^{\text{th}}jth row equal to 1r−1(u(j)−m)\frac{1}{\sqrt{r-1}}\left(u^{(j)} - m\right)r−1​1​(u(j)−m), then we see that C=AA⊤C = AA^\topC=AA⊤. The matrix AAA is full rank if the vectors {u(j)}\{u^{(j)}\}{u(j)} are independent.
