Active Subspaces

An introduction to dimensionality reduction via active subspaces.

Published

30 May 2024

Motivation

Consider some (deterministic) function of interest f:X⊆Rd→R. f: \mathcal{X} \subseteq \mathbb{R}^d \to \mathbb{R}. f:X⊆Rd→R. While this is a generic setup, the common motivation is that fff conceptualizes a computationally costly black-box simulation. There are many things one might wish to do with such a simulation:

  • Minimize or maximize fff [optimization]
  • Integrate fff over some domain of interest [numerical integration]
  • Find input values x∈Rdx \in \mathbb{R}^dx∈Rd to make f(x)f(x)f(x) agree with some observed data [calibration]
  • Assess how the the function outputs f(x)f(x)f(x) vary as the inputs xxx change [sensitivity analysis]
  • Fit a cheaper-to-evaluate surrogate model that approximates fff [emulation/surrogate modeling]
  • Identify subsets of X\mathcal{X}X that result desirable or undesirable outputs [contour/excursion set estimation]

All of these tasks are made more difficult when the number of inputs ddd is large. However, it has been routinely noted that such high-dimensional settings often exhibit low-dimensional structure; i.e., low “intrinsic dimensionality.” This is to say that fff varies primarily on some low-dimensional manifold embedded in the higher-dimensional ambient space X\mathcal{X}X. The active subspace method assumes that the manifold is a linear subspace, and seeks to identify this subspace using gradient evaluations of fff.

Ridge Functions

Before introducing the method of active subspaces, we start to build some intuition regarding the notion of low-dimensional subspace structure in the map fff. The extreme case of this idea is a function of the form \begin{align} f(x) &= g(Ax), &&A \in \mathbb{R}^{r \times d}, \end{align} where r<dr < dr<d. A function of this type is sometimes called a ridge function. The ridge assumption is a rather stringent one, as it implies that fff truly only varies on a subspace of X\mathcal{X}X, and is constant otherwise. Indeed, let x,x~∈Xx, \tilde{x} \in \mathcal{X}x,x~∈X such that x~∈null(A)\tilde{x} \in \text{null}(A)x~∈null(A) (the null space of AAA). Then \begin{align} f(\tilde{x}) &= g(A\tilde{x}) = g(0) \newline f(x + \tilde{x}) &= g(Ax + A\tilde{x}) = g(Ax). \end{align} We can consider the input space as partitioned via the direct sum X=null(A)⊕row(A)\mathcal{X} = \text{null}(A) \oplus \text{row}(A)X=null(A)⊕row(A) (using the fact that the orthogonal complement of the null space is the row space). The ridge function fff is responsive only when xxx changes in row(A)\text{row}(A)row(A), and not at all when xxx only changes in null(A)\text{null}(A)null(A). Although we have not defined the concept yet, it seems intuitive that row(A)\text{row}(A)row(A) would represent the “active subspace” in this case.

Setup and Assumptions

As noted above, the assumption that a function has ridge structure is quite strong. The notion of active subspaces invokes looser assumptions, assuming that, on average, the function primarily varies along a subspace and varies significantly less on its orthogonal complement. To make precise what we mean by “on average”, we introduce a probability measure μ\muμ on X\mathcal{X}X. The choice of μ\muμ will be dictated by the problem at hand. Given that fff may be highly nonlinear, the question of identifying a subspace that on average captures the majority of the variation is not a simple one. The active subspace method addresses this challenge using gradient information, and thus we assume going forward that fff is differentiable. We denote the gradient of fff at an input x∈Xx \in \mathcal{X}x∈X by ∇f(x):=[D1f(x),…,Ddf(x)]⊤∈Rd. \nabla f(x) := \left[D_1 f(x), \dots, D_d f(x) \right]^\top \in \mathbb{R}^d. ∇f(x):=[D1​f(x),…,Dd​f(x)]⊤∈Rd. On a notational note, for a function ϕ:Rn→Rm\phi: \mathbb{R}^n \to \mathbb{R}^mϕ:Rn→Rm we use Dϕ(x)D\phi(x)Dϕ(x) to denote the m×nm \times nm×n Jacobian matrix. Therefore, when applied to the scalar-valued function fff, we have the relation ∇f(x)=Df(x)⊤\nabla f(x) = Df(x)^\top∇f(x)=Df(x)⊤. By observing gradient evaluations ∇f(xi)\nabla f(x_i)∇f(xi​) at a set of inputs x1,…,xnx_1, \dots, x_nx1​,…,xn​ sampled from μ\muμ, we can get a sense of how the function varies along each coordinate direction, on average. However, it may be the case that the function varies most significantly on a subspace not aligned with the standard coordinate directions. Thus, the idea of active subspaces is to utilize the information in these gradient observations to try to find such a subspace; that is, to find linear combinations of the the coordinate directions along which fff exhibits the largest variation. At this point, the idea here should be sounding quite reminiscent of principal components analysis (PCA). Indeed, there are close ties between the two methods. We therefore briefly review PCA in order to better motivate the active subspace algorithm.

Motivating Active Subspaces via PCA

Doing PCA on the Gradient

In seeking active subspaces, we might try to perform PCA directly on XXX in order to capture its dominant directions of variation. However, this approach fails to take into account the function fff at all. The input xxx may vary significantly (with respect to the measure μ\muμ) in a certain direction, but this variation might induce essentially no change in the function output. We seek a method that accounts for both the variation in xxx induced by μ\muμ, as well as the action of the map f(x)f(x)f(x). In fact, it is common to assume that μ\muμ has been centered and normalized such that \begin{align} &\mathbb{E}[x] = 0, &&\text{Cov}[x] = I. \end{align} This means that the dimensions of xxx are already uncorrelated; there is nothing to be gained by performing PCA directly on xxx. However, there still may be low-dimensional structure to exploit in the map fff.

As noted above, the gradient ∇f(x)\nabla f(x)∇f(x) provides information on the local variation in the function along the standard coordinate axes. Alluding to the PCA setting, we can think of these directions of variation as ddd “features”, with the goal of finding linear combinations of the features that explain the majority of the variation. Let’s go with this, taking ∇f(x)\nabla f(x)∇f(x) to be the target for a PCA analysis. Since we’re considering starting with an assumed distribution x∼μx \sim \mux∼μ, it makes sense to apply the distributional form of PCA, as summarized above. Defining the Active Subspace

While I find the PCA comparison quite useful for motivating the method of active subspaces, there is a risk of pushing the analogy too far. In particular, we emphasize that the goal here is not to construct an accurate low-rank approximation of the gradient, but rather to approximate f(x)f(x)f(x) by a function that looks like g(Ax)g(Ax)g(Ax). To provide some initial justification for this latter goal, we show that, on average, fff varies more along the directions defined by the dominant eigenvectors. We let Dvf(x):=∇f(x)⊤vD_v f(x) := \nabla f(x)^\top vDv​f(x):=∇f(x)⊤v denote the directional derivative of fff in the direction vvv.

\begin{align} \lambda_j &= \langle C v_j, v_j \rangle \newline &= v_j^\top \mathbb{E}\left[\nabla f(x) \nabla f(x)^\top\right] v_j \newline &= \mathbb{E}\left[ v_j^\top \nabla f(x) \nabla f(x)^\top v_j \right] \newline &= \mathbb{E}\left[ (\nabla f(x)^\top v_j)^2 \right] \newline &= \mathbb{E}\left[(D_{v_j} f(x))^2 \right] \qquad \blacksquare \end{align} In words, this result says that, on average (with respect to μ\muμ), the squared derivative in the direction of vjv_jvj​ is given by the eigenvalue λj\lambda_jλj​. Understanding the Low-Dimensional Subspace

Having defined the active subspace, we now begin to investigate its basic properties.
