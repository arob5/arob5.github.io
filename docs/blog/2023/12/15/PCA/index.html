Part 1: Formulating and Solving the PCA Optimization Problem

Setup and Notation

Suppose that we have data x1,…,xN∈RD\mathbf{x}_1, \dots, \mathbf{x}_N \in \mathbb{R}^Dx1​,…,xN​∈RD, stacked into the rows of a matrix X∈RN×DX \in \mathbb{R}^{N \times D}X∈RN×D. Our task is to find a subspace of smaller dimension R<DR < DR<D such that the projection of the data points onto the subspace retains as much information as possible. By restricting our attention to orthonormal bases for the low-dimensional subspace, we reduce the problem to finding a set of orthonormal basis vectors \begin{align} &\mathbf{b}_1, \dots, \mathbf{b}_R \in \mathbb{R}^D, &&\langle \mathbf{b}_r, \mathbf{b}_s \rangle = \delta_{r,s} \end{align} Define B∈RD×RB \in \mathbb{R}^{D \times R}B∈RD×R to be the matrix with rthr^{\text{th}}rth column equal to br\mathbf{b}_rbr​. The subspace generated by the basis BBB is given by span(B):=span(b1,…,bR). \text{span}(B) := \text{span}(\mathbf{b}_1, \dots, \mathbf{b}_R). span(B):=span(b1​,…,bR​). Throughout this post I will abuse notation by referring to the matrix BBB when actually talking about the set of vectors {b1,…,bR}\{\mathbf{b}_1, \dots, \mathbf{b}_R\}{b1​,…,bR​}. Since there is no a priori reason to assume that the data is centered, we should also allow for the subspace to be shifted by some intercept w0∈RD\mathbf{w}_0 \in \mathbb{R}^Dw0​∈RD, resulting in the affine space w0+span(B)={w0+∑r=1Rwrbr:w1,…,wR∈R}. \mathbf{w}_0 + \text{span}(B) = \left\{\mathbf{w}_0 + \sum_{r=1}^{R} w_r \mathbf{b}_r : w_1, \dots, w_R \in \mathbb{R} \right\}. w0​+span(B)={w0​+r=1∑R​wr​br​:w1​,…,wR​∈R}. Loosely speaking, the task is to find the basis BBB, intercept w0\mathbf{w}_0w0​, and pointwise weights w1,…,wN∈RR\mathbf{w}_1, \dots, \mathbf{w}_N \in \mathbb{R}^Rw1​,…,wN​∈RR such that \begin{align} \mathbf{x}_n &\approx \mathbf{w}_0 + \sum_{r=1}^{R} (\mathbf{w}_n)_r \mathbf{b}_r, &&\forall n=1,\dots,N \newline &= \mathbf{w}_0 + B\mathbf{w}_n. \end{align} To formalize this notion, PCA measures the error in the above approximation using Euclidean distance, averaged over the NNN data points. To further sumplify notation, we stack the wn\mathbf{w}_nwn​ in the columns of a matrix W∈RR×NW \in \mathbb{R}^{R \times N}W∈RR×N. With all of this notation established, we can state that PCA solves the optimization problem argminB,W,w0∑n=1N∥xn−(w0+Bwn)∥22, \text{argmin}_{B, W, \mathbf{w}_0} \sum_{n=1}^{N} \lVert \mathbf{x}_n - (\mathbf{w}_0 + B\mathbf{w}_n) \rVert_2^2, argminB,W,w0​​n=1∑N​∥xn​−(w0​+Bwn​)∥22​, where the basis BBB is constrained to be orthonormal. As we will see, this optimization naturally breaks down into two distinct problems which can be solved sequentially:

  1. Given the basis BBB and intercept w0\mathbf{w}_0w0​, find the optimal basis coefficients wn\mathbf{w}_nwn​ corresponding to each data point xn\mathbf{x}_nxn​.
  2. Find the optimal basis and intercept. Part of the popularity of PCA stems from the fact that both problems can be solved in closed-form. Let us consider both problems in turn.

Optimizing the Basis Coefficients

Let us first consider w0\mathbf{w}_0w0​ and BBB to be fixed, meaning that we are fixing an affine subspace of dimension RRR. We seek to find the optimal way to represent the data XXX in this lower-dimensional space. As we will show, the Euclidean objective used by PCA implies that this problem reduces to straightforward orthogonal projection. For now, let xnc:=xn−w0\mathbf{x}^c_n := \mathbf{x}_n - \mathbf{w}_0xnc​:=xn​−w0​ denote the centered data points (we will deal with the intercept shortly). We are thus considering the problem argminW∑n=1N∥xnc−Bwn∥22 \text{argmin}_{W} \sum_{n=1}^{N} \lVert \mathbf{x}^c_n - B\mathbf{w}_n \rVert_2^2 argminW​n=1∑N​∥xnc​−Bwn​∥22​ Observe that wn\mathbf{w}_nwn​ only appears in the nthn^{\text{th}}nth term of the sum, meaning that we can consider each summand independently,
argminwn∥xnc−Bwn∥22. \text{argmin}_{\mathbf{w}_n} \lVert \mathbf{x}^c_n - B\mathbf{w}_n \rVert_2^2. argminwn​​∥xnc​−Bwn​∥22​. In words, we seek the linear combination of the basis vectors BBB that results in minimal Euclidean distance from xn\mathbf{x}_nxn​; this is a standard orthogonal projection problem from linear algebra. Since the basis vectors are orthonormal, the optimal projection coefficients are given by \begin{align} &(\mathbf{w}_n)_r = \langle \mathbf{x}_n, \mathbf{b}_r \rangle, &&\mathbf{w}_n = B^\top \mathbf{x}_n \end{align} which can be written succinctly for all data points as W⊤=XcB, W^\top = X^c B, W⊤=XcB, with XcX^cXc denoting the centered data matrix with rows set to the (xnc)⊤(\mathbf{x}^c_n)^\top(xnc​)⊤.

Optimizing the Basis and Intercept

In the previous section, we saw that for a fixed basis and intercept, optimizing the basis weights reduced to an orthogonal projection problem. In this section we show that for fixed weights, optimizing the basis reduces to solving a sequence of eigenvalue problems. To be clear, we are now considering the problem

argminB∑n=1N∥xn−Bwn∥22, \text{argmin}_{B} \sum_{n=1}^{N} \lVert \mathbf{x}_n - B\mathbf{w}_n \rVert_2^2, argminB​n=1∑N​∥xn​−Bwn​∥22​, where we are additionally treating w0\mathbf{w}_0w0​ as fixed for the time being.

An Alternative Approach: The Eckart-Young Theorem

Armed with the Eckart-Young Theorem, we can derive the same result with much less work. The key is to re-write the reconstruction error as a a matrix approximation problem using the Frobenius norm. We have

∑n=1N∥xn−Bwn∥22=∥X−WB⊤∥22, \sum_{n=1}^{N} \lVert \mathbf{x}_n - B\mathbf{w}_n \rVert_2^2 = \lVert X - WB^\top \rVert_2^2, n=1∑N​∥xn​−Bwn​∥22​=∥X−WB⊤∥22​, where the function ∥⋅∥2\lVert \cdot \rVert_2∥⋅∥2​ denotes the Frobenius norm when passed a matrix. The PCA optimization problem can then be written as the matrix approximation problem argminB,W∥X−WB⊤∥22, \text{argmin}_{B, W} \lVert X - WB^\top \rVert_2^2, argminB,W​∥X−WB⊤∥22​, where BBB is constrained to be an orthogonal matrix. This is precisely the problem considered by the Eckart-Young theorem.

In this section, I briefly discuss the spectral norm and eigenvalue problems in finite-dimensional vector spaces, which I utilize above when optimizing the basis BBB in the PCA derivation.
