Sigma Algebra

We recall from the previous post that a univariate Gaussian measure is defined on the Borel σ\sigmaσ-algebra B(R)\mathcal{B}(\mathbb{R})B(R). Analogously, we will define an nnn-dimensional Gaussian measure on the Borel sets B(Rn)\mathcal{B}(\mathbb{R}^n)B(Rn). There are two reasonable approaches to define B(Rn)\mathcal{B}(\mathbb{R}^n)B(Rn), and I want to take a moment to highlight them since the same two options will present themselves when we consider defining the Borel sets over infinite-dimensional spaces.

Option 1: Leverage the Standard Topology on Rn\mathbb{R}^nRn

A Borel σ\sigmaσ-algebra can be defined for any space that comes equipped with a topology; i.e., a collection of open sets. The Borel σ\sigmaσ-algebra is then defined as the smallest σ\sigmaσ-algebra that contains all of these open sets. In the present setting, this means B(Rn):=σ{O⊆Rn:O is open},(1) \mathcal{B}(\mathbb{R}^n) := \sigma\left\{\mathcal{O} \subseteq \mathbb{R}^n : \mathcal{O} \text{ is open} \right\}, \tag{1} B(Rn):=σ{O⊆Rn:O is open},(1) where σ(S)\sigma(\mathcal{S})σ(S) denotes the σ\sigmaσ-algebra generated by a collection of sets S\mathcal{S}S. A nice perspective on B(Rn)\mathcal{B}(\mathbb{R}^n)B(Rn) is that it is the smallest σ\sigmaσ-algebra that ensures all continuous functions f:Rn→Rf: \mathbb{R}^n \to \mathbb{R}f:Rn→R are measurable. We note that this is not a property of Borel σ\sigmaσ-algebras more generally, but one that does hold in the special case of Rn\mathbb{R}^nRn; see this StackExchange post for some details.

Option 2: Product of One-Dimensional Borel Sets

A second reasonable approach is to try extending what we have already defined in one dimension, which means simply taking Cartesian products of one-dimensional Borel sets: B(Rn):=σ{B1×⋯×Bn:Bi∈B(R), i=1,…,n}.(2) \mathcal{B}(\mathbb{R}^n) := \sigma\left\{B_1 \times \cdots \times B_n: B_i \in \mathcal{B}(\mathbb{R}), \ i = 1, \dots, n \right\}. \tag{2} B(Rn):=σ{B1​×⋯×Bn​:Bi​∈B(R), i=1,…,n}.(2) It turns out that the resulting σ\sigmaσ-algebra agrees with that defined in option 1, so there is no ambiguity in the notation.

Definition: One-Dimensional Projections

With the σ\sigmaσ-algebra defined, we now consider how to define a Gaussian measure μ\muμ on the measurable space (Rn,B(Rn))(\mathbb{R}^n, \mathcal{B}(\mathbb{R}^n))(Rn,B(Rn)). We will explore a few different equivalent definitions, starting with this: a measure is Gaussian if all of its one-dimensional projections are (univariate) Gaussians.

Definition. A probability measure μ\muμ defined on the Borel measurable space (Rn,B(Rn))(\mathbb{R}^n, \mathcal{B}(\mathbb{R}^n))(Rn,B(Rn)) is called Gaussian if, for all linear maps ℓ∈(Rn)∗\ell \in (\mathbb{R}^n)^*ℓ∈(Rn)∗, the pushforward measure μ∘ℓ−1\mu \circ \ell^{-1}μ∘ℓ−1 is Gaussian on (R,B(R))(\mathbb{R}, \mathcal{B}(\mathbb{R}))(R,B(R)).

Transformation of Standard Gaussian Random Variables

In this section we provide yet another characterization of Gaussian measures. We consider a generative perspective, whereby a Gaussian random vector $X \in \mathbb{R}^n$ arises via a linear transformation of $n$ iid $\mathcal{N}(0,1)$ random variables.

Proposition. Let $Z_1, \dots, Z_n$ be iid $\mathcal{N}(0, 1)$ random variables stacked into the column vector $Z \in \mathbb{R}^n$. Then, for any fixed vector $m \in \mathbb{R}^n$ and matrix $A \in \mathbb{R}^{n \times n}$, the random variable given by $$ X := m + AZ \tag{6} $$ has a Gaussian distribution $\mathcal{N}(m, AA^\top)$. Conversely, let $X \in \mathbb{R}^n$ be a Gaussian random variable. Then there exists a vector $m \in \mathbb{R}^n$ and matrix $A \in \mathbb{R}^{n \times n}$ such that $X = m + AZ$.

Another way to think about this is that we have defined a transport map T:Rn→RnT: \mathbb{R}^n \to \mathbb{R}^nT:Rn→Rn such that \begin{align} T(Z) &= X, &&\text{where } T(z) = m + Az. \end{align} That is, we feed in vectors with iid standard Gaussian components, and get out vectors with distribution N(m,AA⊤)\mathcal{N}(m, AA^\top)N(m,AA⊤). This is a very practical way to look at multivariate Gaussians, immediately providing the basis for a sampling algorithm. Indeed, suppose we want to draw iid samples from the distribution N(m,C)\mathcal{N}(m, C)N(m,C). Then the above proposition gives us a way to do so, provided that we can (1) draw univariate N(0,1)\mathcal{N}(0,1)N(0,1) samples; and (2) factorize the matrix CCC as C=AA⊤C = AA^\topC=AA⊤ for some AAA. This procedure is summarized in the below corollary.

Corollary. The following algorithm produces a sample from the distribution N(m,C)\mathcal{N}(m, C)N(m,C).
1. Draw nnn iid samples Zi∼N(0,1)Z_i \sim \mathcal{N}(0,1)Zi​∼N(0,1) and stack them in a column vector ZZZ.
2. Compute a factorization C=AA⊤C = AA^\topC=AA⊤.
3. Return m+AZm + AZm+AZ.
Repeating steps 1 and 3 will produce independent samples from N(m,C)\mathcal{N}(m, C)N(m,C) (the matrix factorization need not be re-computed each time).

As for the factorization, the Cholesky decomposition is a standard choice. The eigendecomposition provides another option, since C=UDU⊤=UD1/2D1/2U⊤=(UD1/2)(UD1/2)⊤ C = UDU^\top = UD^{1/2} D^{1/2} U^\top = (UD^{1/2})(UD^{1/2})^\top C=UDU⊤=UD1/2D1/2U⊤=(UD1/2)(UD1/2)⊤ so setting A:=UD1/2A := UD^{1/2}A:=UD1/2 does the trick. Note that CCC is positive semidefinite so DDD is just a diagonal matrix with nonnegative values on the diagonal.
