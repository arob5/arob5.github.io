Pseudo-Marginal MCMC

Motivated by Bayesian inference, I introduce the pseudo-marginal approach to MCMC and then discuss why is works from a more generic perspective.

Published

28 September 2024

Pseudo-marginal Markov chain Monte Carlo (MCMC) is a variant of the Metropolis-Hastings algorithm that works without the ability to evaluate the unnormalized target density, so long as an unbiased sample of this density can be obtained for any input. In this post, we motivate the algorithm by considering a problem of Bayesian inference where the likelihood function is intractable. We then take a step back to understand why the algorithm works, and discuss the method from a more generic and rigorous viewpoint.

Pseudo-Marginal MCMC for Bayesian Inference

We start by considering a standard problem of Bayesian inference for a parameter of interest u∈Uu \in \mathcal{U}u∈U. However, suppose we face a situation where L(u)L(u)L(u) is intractable in the sense that it does not admit an analytic expression that can be computed for any uuu. Suppose, though, that we can draw an unbiased sample of the quantity L(u)L(u)L(u) for any input uuu; that is, \begin{align} &\ell \sim P(u, \cdot), &&\mathbb{E}[\ell] = L(u), \tag{2} \end{align} where P(u,⋅)P(u,\cdot)P(u,⋅) is a probability measure on the sample space [0,∞)[0, \infty)[0,∞) for each u∈Uu \in \mathcal{U}u∈U (formally, we can think of PPP as a Markov kernel). A More Generic and Formal Perspective

The above idea of course extends beyond the Bayesian example. In this section, we discuss the pseudo-marginal algorithm from a more generic perspective, and fill in some of the measure-theoretic details. Let’s assume Π\PiΠ is some generic target distribution on a measurable space (U,B(U))(\mathcal{U}, \mathcal{B}(\mathcal{U}))(U,B(U)). We write B(U)\mathcal{B}(\mathcal{U})B(U) to denote the Borel σ\sigmaσ-algebra; that is, the σ\sigmaσ-algebra generated by the open sets of U\mathcal{U}U. We assume Π\PiΠ admits a density (i.e., Radon-Nikodym derivative) π\piπ with respect to some reference measure ν\nuν. The density π\piπ need not be normalized. All densities considered throughout this section will be with respect to the same reference measure ν\nuν. As before, we consider π(u)\pi(u)π(u) intractable, but assume we can draw samples from an unbiased estimator. We could define P(u,⋅)P(u,\cdot)P(u,⋅) as before such that samples drawn from P(u,⋅)P(u,\cdot)P(u,⋅) are unbiased with respect to π(u)\pi(u)π(u). This seems to be a roundabout way to go around this, but for the purposes of analysis it turns out to be convenient.
